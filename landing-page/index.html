Remote Control via HP
Scan QR code, buka di browser HP. Navigasi slide prev/next, blank screen, dan cari Alkitab langsung dari genggaman tangan Anda.

Stage Display
Jendela terpisah untuk musisi: lirik saat ini (besar), preview berikutnya, pesan operator, dan countdown timer terintegrasi.

Countdown Timer
Timer hitung mundur dengan durasi atau target waktu. Background custom, tampil di projektor dan stage display bersamaan.

Export semua data ke ZIP, restore di komputer lain. Deteksi duplikat otomatis. Backup selektif: lagu saja, media saja, atau semua.
